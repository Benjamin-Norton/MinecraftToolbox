Welcome to Fasguy's Minecraft Toolbox

The toolbox provides multiple different tools to aid in or alter your Minecraft game experience
  • Loot-Table Randomizer
    The Loot-Table Randomizer randomly switches the loot-tables (i.e. mob drops, block drops, etc.) around.
  • Crafting-Recipe Randomizer
    The Crafting-Recipe Randomizer randomly switches the crafting recipes of all items around.
  • Vanilla Tweaks
    Vanilla Tweaks provides multiple data and resource packs that alter the game experience in subtle ways.
  • Pixel Perfect
    Pixel Perfect provides gameplay-altering Minecraft Bedrock Edition Add-Ons.
