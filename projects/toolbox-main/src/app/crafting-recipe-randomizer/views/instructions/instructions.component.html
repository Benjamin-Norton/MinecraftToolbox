Installing during world creation (Minecraft v1.16+)

  1. On the main menu go to Singleplayer -> Create New World
  2. Go into the Data Packs menu
  3. Drag and drop the downloaded data pack into Minecraft
  4. Confirm that you want to import the data pack
  5. Click on the icon of the data pack to move it from the Available list to the Selected list
  6. Click Done
  7. Generate the world

Installing into an existing Minecraft world

  1. Go into your '.minecraft' directory. (Click here for help)
  2. Go into 'saves' and then into the world's folder, where you want the drops to be randomized.
  3. Go into 'datapacks' and place the generated .zip file in this folder.
  4. Load the world in Minecraft
